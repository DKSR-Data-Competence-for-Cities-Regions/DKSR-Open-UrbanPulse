Possible queries:

/sensordata?sid=<UUID>&since=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>&until=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>
Returns all events for the given SID between the given timestamps.
/sensordata?sid=<UUID>
Returns the latest event for the given SID.
/sensordata?eventtype=<UUID>&since=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>&until=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>
Returns all events for all sensors with the given eventtype-ID between the given timestamps.
/sensordata?eventtype=<UUID>
Returns the latest event for each sensor with the given eventtype-ID.
/sensordata?eventtypeName=<String>&since=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>&until=<yyyy-MM-dd'T'HH:mm:ss.SSSXX>
Returns all events for all sensors with the given eventtype-name between the given timestamps.
/sensordata?eventtypeName=<String>
Returns the latest event for each sensor with the given eventtype-name.
